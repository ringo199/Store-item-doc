<h1>果蔬小屋项目开发说明</h1>
<ul>
<li>整个项目中，包含了系统功能和需求说明书，Server接口文档摘要、Frontend设计需求说明、Android客户端的需求说明，iOS客户端的需求说明、商家管理端的需求说明，以及系统数据库设计。</li>
